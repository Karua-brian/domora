# Domora Learning Philosophy

Learning software engineering begins with understanding reality.

Programming languages are tools for expressing models.

Frameworks are tools for implementing architectures.

Architectures are tools for protecting business rules.

Business rules describe how reality operates.

Therefore:

Reality

↓

Understanding

↓

Mental Models

↓

Architecture

↓

Programming

↓

Technology

We learn from the top down.

Not because code is unimportant.

Because code without understanding produces fragile software.
